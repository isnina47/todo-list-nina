# React Todo List Demo

這是一個使用 React Hooks（Function Component）實作的 Todo List 專案，具備以下功能：

1. 新增待辦事項
2. 勾選完成／未完成（文字加上刪除線）
3. 刪除事項
4. 自動捲動至最底
5. 顯示完成進度條（整數百分比）
6. 切換完成項目是否顯示在下方

## 🔧 使用技術

- React 18
- Function Components + Hooks（useState、useEffect）
- Tailwind
- Git & GitHub
